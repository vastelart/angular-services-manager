## ANGULAR SERVICES MANAGER

Развернуть у себя:
---
1. git clone https://github.com/vastelart/angular-services-manager.git servicemanager  
2. cd servicemanager  
3. bower i  
4. npm i  
5. gulp serve — запустит php-сервер 
6. gulp build — соберет продакшен-билд в папку dist
